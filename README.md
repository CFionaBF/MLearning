# Task Description

第一步：确定核心需求
明确目的地、预估出发日期、同行人数以及大致的人均预算。

第二步：搜索往返大交通
前往 [携程] 或 [飞猪] 搜索航班，关注特定航司的会员日折扣。

第三步：寻找住宿方案
结合地理位置（如靠近地铁站或景区）在平台上比价，选择评分 4.5 以上的酒店。

第四步：记录你的计划
打开https://docs.google.com/document/d/19dpy0RDz8d4MC2RHjg_gFfDunMWRKtQMTBixBW06P64/edit?tab=t.0, 将计划写入其中。
